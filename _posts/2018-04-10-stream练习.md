<p>遍历list里的map</p>

```
        List<Map<String,String>>cdrList = new ArrayList<>();
        Map<String,String>m = new HashMap<>();
        m.put("1","2");
        cdrList.add(m);
        Map<String,String>mm = new HashMap<>();
        mm.put("3","4");
        cdrList.add(mm);
        //这两个都可以，待会补上差别
        cdrList.stream().forEach(map -> {
            map.forEach((k,v) -> System.out.println(k+ "" + v));
        });
        cdrList.forEach(map -> {
            map.forEach((k,v) -> System.out.println(k+ "" + v));
        });
```
